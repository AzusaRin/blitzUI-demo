## Collapse Attributes


| 参数       | 说明              | 类型      | 可选值 |  默认值  |
|----------|-----------------|---------|-----|:-----:|
| selected | 绑定值，默认打开页的 name | Array   | —   |   —   |
| single   | 设置单页打开          | Boolean | —   | false |


---

## Collapse-item Attributes

| 参数    | 说明                 | 类型     | 可选值 | 默认值 |
|-------|--------------------|--------|:---:|:---:|
| title | 页标题名               | String |  —  |  —  |
| name  | 与页绑定值对应的标识符，表示页的别名 | String |  —  |  —  |
